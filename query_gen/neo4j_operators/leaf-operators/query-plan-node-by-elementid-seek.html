Node By ElementId Seek

The NodeByElementIdSeek operator reads one or more nodes by ID from the node store, specified via the function elementId().

Example 9. NodeByElementIdSeek
Query
PROFILE
MATCH (n)
WHERE elementId(n) = 0
RETURN n
