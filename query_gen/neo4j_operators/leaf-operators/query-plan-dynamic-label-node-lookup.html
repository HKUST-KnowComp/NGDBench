Dynamic Label Node Lookup
Introduced in 2025.08

Allows Cypher to use token lookup indexes when planning queries using dynamic node labels.

Example 45. DynamicLabelNodeLookup
Query
PROFILE
WITH "Person" AS label
MATCH (people:$(label))
RETURN people.name
