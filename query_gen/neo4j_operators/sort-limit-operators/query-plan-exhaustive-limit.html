Exhaustive Limit

The ExhaustiveLimit operator is similar to the Limit operator but will always exhaust the input. Used when combining LIMIT and updates

Example 104. ExhaustiveLimit
Query
PROFILE
MATCH (p:Person)
SET p.seen = true
RETURN p
LIMIT 3
