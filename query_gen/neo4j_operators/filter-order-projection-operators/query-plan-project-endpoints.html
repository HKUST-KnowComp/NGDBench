Project Endpoints

The ProjectEndpoints operator projects the start and end node of a relationship.

Example 93. ProjectEndpoints
Query
PROFILE
CREATE (n)-[p:KNOWS]->(m)
WITH p AS r
MATCH (u)-[r]->(v)
RETURN u, v
