Drop Constraint

The DropConstraint operator removes a constraint using the name of the constraint, no matter the type.

Example 129. DropConstraint
Query
PROFILE
DROP CONSTRAINT uniqueness IF EXISTS
Query Plan
Planner ADMINISTRATION
